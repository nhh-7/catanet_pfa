# 基于内容感知渐进聚焦机制的轻量级图像超分辨率网络 CAPA-SR 开题报告

## 摘要

单幅图像超分辨率旨在从低分辨率图像中恢复高分辨率细节，是计算机视觉与图像处理领域的重要基础问题，在遥感成像、医学影像、视频监控、移动终端视觉增强等场景中具有广泛应用价值。近年来，基于深度学习的超分辨率方法在重建质量上取得了显著进展，但在轻量级场景下仍面临三类突出矛盾：一是固定局部窗口难以建模跨区域的长程纹理依赖；二是稠密注意力计算中存在大量无关交互，容易带来噪声传播和计算冗余；三是单尺度特征建模难以兼顾结构连续性与高频纹理恢复。针对上述问题，本课题拟提出一种面向轻量级图像超分辨率任务的内容感知渐进聚焦网络 `CAPA-SR`。该方法以内容组织、关键连接筛选和低到高递进重建为核心思路，设计动态语义原型路由、渐进聚焦稀疏注意力以及低到高多级协同重建三类关键机制，在有限参数量和计算量预算下提升复杂结构场景中的重建质量。

本课题将依托现有轻量级超分辨率代码框架开展研究，围绕 `DF2K` 训练集与 `Set5`、`Set14`、`BSD100`、`Urban100`、`Manga109` 等标准测试集开展系统实验，并使用 `PSNR`、`SSIM`、参数量、`FLOPs` 与推理延迟等指标综合评估模型性能。已有阶段性结果表明，`CAPA-SR` 在 `x4` 设置下以 `567K` 参数、`46.1G FLOPs` 和 `85.1 ms` 延迟，在 `Urban100` 与 `Manga109` 上分别达到 `27.24 dB` 和 `31.55 dB`，相较于基线 `CATANet` 在复杂结构场景中展现出较好的性能与效率平衡。预期本研究可为轻量级图像超分辨率模型的结构设计提供新的方法路径。

**关键词：** 图像超分辨率；轻量级网络；内容感知路由；稀疏注意力；多级重建

## 1. 研究背景与意义

图像超分辨率技术通过学习低分辨率图像与高分辨率图像之间的映射关系，实现对图像细节、纹理与边缘结构的重建。随着高清显示、移动拍摄与边缘智能设备的快速普及，模型在实际应用中不仅需要具备较高重建精度，还需要满足参数量小、推理快、部署友好的工程要求。因此，轻量级超分辨率已经成为图像恢复方向的重要研究分支。

从方法演进来看，早期工作多采用卷积神经网络逐步提升超分辨率精度，如 `SRCNN`、`EDSR`、`RCAN` 等方法在特征表达和残差建模方面取得了显著成效。但随着分辨率提升任务对长程依赖与重复纹理建模需求的增强，仅依赖局部卷积感受野往往难以充分利用远距离相似区域之间的信息。随后，`SwinIR` 等基于 Transformer 的方法将自注意力机制引入图像恢复任务，使模型能够更有效地建模长距离相关性。然而，标准自注意力计算复杂度较高，轻量化版本通常又依赖固定窗口划分，这会限制跨区域相似纹理之间的直接交互。

近年来，内容感知聚合和稀疏注意力逐渐成为轻量级超分辨率的重要改进方向。以 `CATANet` 为代表的方法尝试基于内容相似性组织 Token，以提升长程建模能力；以 `PFT` 为代表的方法则通过渐进聚焦思想抑制无效交互，以改善注意力效率。然而，现有方法仍然存在明显不足：其一，内容组织往往依赖静态或近似静态中心，缺乏足够的图像自适应性；其二，稀疏连接筛选更多聚焦于单层或单级建模，尚未与层次化重建过程充分结合；其三，轻量级模型在结构先验与纹理补偿之间仍缺少统一的递进式设计。

基于此，本课题拟围绕“内容组织、连接筛选、层次重建”三者协同这一主线展开研究。若能在轻量级预算下提升模型对复杂结构、规则纹理和高频边缘的恢复能力，不仅可以提升方法在标准学术基准上的表现，也能够为移动端、实时视觉增强等受限计算场景提供更具实用价值的解决方案。因此，本课题具有较强的理论研究意义与工程应用价值。

## 2. 国内外研究现状

当前图像超分辨率研究大致可以分为卷积驱动方法、Transformer 驱动方法，以及内容感知或稀疏注意力增强方法三条主线。

第一类方法以卷积网络为主。`IMDN`、`RFDN`、`CARN` 等轻量级模型通过信息蒸馏、特征重用和级联结构在较小参数量下取得了较好的重建效果。这类方法具有实现简单、推理稳定的优点，但由于主要依赖局部卷积堆叠，对远距离重复纹理和全局结构模式的捕获能力相对有限。

第二类方法将 Transformer 引入超分辨率任务。`SwinIR`、`ELAN-light`、`ATD-light`、`IPG-Tiny` 等方法在建模长程依赖方面表现突出，尤其在复杂纹理场景中具有优势。但为了控制计算复杂度，很多轻量化 Transformer 仍依赖固定窗口、局部分组或预定义稀疏模式，这使模型对跨窗口重复结构的感知受到约束，且在低算力预算下容易出现计算负担较高的问题。

第三类方法关注内容感知分组与稀疏交互。`CATANet` 通过内容感知 Token 聚合模块组织长程内容相似 Token，并利用组内自注意力与组间交叉注意力提升轻量级超分辨率性能，说明内容组织机制对于复杂结构重建具有重要作用。`PFT` 则提出渐进聚焦注意力，通过注意力继承与焦点约束机制降低无关连接带来的计算冗余，证明注意力并非越稠密越有效，聚焦关键区域同样能够获得更好的性能和效率平衡。

尽管现有方法已经取得一系列成果，但综合来看仍存在以下研究空白。

1. 现有内容感知聚合方法往往依赖固定中心或较强的静态先验，面对测试图像中的新纹理分布时自适应能力不足。
2. 稀疏注意力机制多关注连接压缩本身，但较少结合超分辨率“结构优先、纹理后补”的重建规律进行层次化建模。
3. 许多轻量级方法虽在平滑数据集上取得稳定收益，但在 `Urban100`、`Manga109` 等复杂结构数据集上仍难兼顾结构连续性、纹理清晰度与计算代价。

因此，从国内外研究进展来看，围绕图像内容动态组织、关键连接渐进聚焦以及低到高多级重建展开统一设计，仍然具有较大的研究空间。本课题拟在这一方向上进行系统探索。

## 3. 研究目标与拟解决的关键问题

### 3.1 研究目标

本课题的总体目标是设计一种面向轻量级单幅图像超分辨率任务的内容感知渐进聚焦网络 `CAPA-SR`，在较低参数量与计算量预算下提升模型对复杂结构、重复纹理和高频边缘的重建能力，形成兼顾性能、效率与可解释性的轻量级超分辨率方法。

具体目标包括：

1. 构建能够随输入内容动态变化的语义路由机制，提升跨区域相似纹理的组织与交互能力。
2. 构建结合当前相似性与历史注意力先验的稀疏注意力机制，降低无关 Token 交互带来的噪声与冗余计算。
3. 构建从低频到高频递进的信息注入链路，使模型更符合超分辨率的层次化重建规律。
4. 在标准公开数据集上系统验证所提方法在重建精度、复杂度和推理效率上的综合优势。

### 3.2 拟解决的关键问题

围绕上述目标，本课题重点解决以下三个关键问题：

1. 如何在不显著增加模型复杂度的前提下，使远距离但语义相似的纹理区域获得有效交互。
2. 如何让注意力计算从“默认全连接”转向“按内容选择关键连接”，并通过动态稀疏机制提升有效计算比例。
3. 如何建立结构到纹理的递进式重建链路，使轻量级模型在保持效率的同时获得更稳定的细节恢复能力。

## 4. 研究内容与方法设计

### 4.1 总体框架

`CAPA-SR` 面向端到端轻量级图像超分辨率重建任务，整体流程为：

`低分辨率输入 -> 浅层特征提取 -> 多个 CAPA Block -> 重建头 -> 超分结果`

其中，`CAPA Block` 是本课题的核心计算单元，主要由四部分组成：

1. `DPR`：动态语义原型路由。
2. `PFSA`：渐进聚焦稀疏注意力。
3. `LMR`：低到高多级协同重建。
4. `LFR`：轻量级前馈细化模块。

该框架的核心思想是先按内容组织 Token，再按重要性筛选连接，最后按照低频到高频的顺序逐级完成结构恢复与纹理补偿。

### 4.2 动态语义原型路由

针对固定窗口难以覆盖跨区域相似纹理的问题，本课题拟为当前输入特征动态生成一组语义原型，并根据 Token 与原型之间的相似性完成内容路由。与传统静态聚类中心不同，该模块强调输入图像自适应性，使原本在空间上相距较远、但在语义或纹理属性上相近的 Token 能够被归入更合理的候选集合中。

该设计预期带来两方面收益：一是打破固定局部建模的空间限制，为远距离重复结构建立交互基础；二是为后续稀疏注意力提供更干净的候选集合，减少不相关 Token 混入同一交互组的概率。

### 4.3 渐进聚焦稀疏注意力

在完成内容路由后，本课题不采用普通组内稠密注意力，而是引入渐进聚焦稀疏注意力。该模块综合利用上一层注意力先验与当前层内容相似性，动态生成稀疏连接掩码，仅保留最关键的连接进行注意力计算。同时，通过组内纯度和分数方差估计动态焦点率 `r`，使模型具备“纯组更密、混组更稀”的自适应聚焦特性。

该模块的研究重点包括：

1. 设计多级注意力继承状态，分别维护低频、中频和高频分支的聚焦先验。
2. 设计基于组内纯度与方差的动态焦点率机制，实现内容感知稀疏化。
3. 研究注意力继承与掩码生成方式对性能、复杂度和数值稳定性的影响。

### 4.4 低到高多级协同重建

超分辨率本质上并非单一尺度的锐化问题，而是一个“先恢复结构，再补足边界，最后强化纹理”的递进过程。基于这一认识，本课题拟将路由后的特征拆分为低频、中频和高频三个层级，分别负责大范围结构关系、区域边界过渡和高频细节恢复，并通过低到高逐级注入方式完成多级重建。

该模块预期实现以下目标：

1. 在低频层级优先学习结构连续性与轮廓先验。
2. 在中频层级增强边界过渡与区域级结构表达。
3. 在高频层级在前序引导下稳定恢复纹理、边缘和重复细节。

### 4.5 研究假设

为保证课题具有明确的可检验性，拟提出以下研究假设：

1. 引入动态语义原型路由后，模型在复杂结构与重复纹理数据集上的重建质量将优于仅采用静态内容聚合的基线模型。
2. 引入渐进聚焦稀疏注意力后，模型可在降低无效计算的同时提升 `Urban100`、`Manga109` 等数据集上的重建精度。
3. 引入低到高多级协同重建后，模型的结构连续性和高频纹理恢复能力将优于单尺度注意力设计。

## 5. 技术路线与实验方案

### 5.1 技术路线

本课题拟按照“基线复现、模块替换、联合优化、系统验证”的路线推进：

1. 以现有轻量级超分辨率框架为基础，复现内容感知主干与重建头。
2. 在主干网络中插入 `CAPA Block`，实现 `DPR`、`PFSA` 与 `LMR` 三个核心模块。
3. 建立多级注意力状态继承机制与全局交互分支，实现深层特征逐层聚焦。
4. 在统一训练配置下完成主实验、消融实验、复杂度实验和可视化分析。
5. 基于实验结果进一步调整原型数量、焦点率范围、层级划分和头部分配策略。

### 5.2 数据集与评价指标

实验计划采用 `DF2K` 作为训练集，其中包含 `DIV2K` 与 `Flickr2K`；采用 `Set5`、`Set14`、`BSD100`、`Urban100`、`Manga109` 作为标准测试集；统一考虑 `x2`、`x3`、`x4` 三种放大倍率。

评价指标包括：

1. 重建质量指标：`PSNR` 与 `SSIM`，统一在 `YCbCr` 的 `Y` 通道上计算。
2. 复杂度指标：模型参数量、`FLOPs`。
3. 部署指标：在 `RTX 4090` 上测得的推理延迟。

### 5.3 训练与实现设置

根据现有实验设计，模型将采用 `64 x 64` 的低分辨率 patch、`32` 的 batch size，并使用 `AdamW` 优化器进行训练，初始学习率设置为 `5e-4`，采用 cosine decay 策略。主实验训练 `500K` iteration，消融实验统一训练 `300K` iteration。数据增强包括随机翻转、随机旋转和随机 RGB shuffle。

在结构配置上，计划采用 `head_split = [2, 1, 1]` 作为默认多级分配方式，并设置 `r` 的动态范围为 `[0.25, 0.75]`，注意力继承权重 `beta` 默认为 `0.5`。该配置兼顾了高频建模能力、中低频结构补偿能力与整体复杂度控制。

### 5.4 对比实验与消融实验

为充分验证方法有效性，本课题拟设计如下实验。

1. 与 `CARN`、`IMDN`、`RFDN`、`OSFFNet`、`SwinIR-light`、`ELAN-light`、`ATD-light`、`IPG-Tiny`、`CATANet` 和 `PFT-light` 进行主实验对比。
2. 开展核心模块消融，比较仅基线、加入 `DPR`、加入 `PFSA`、加入 `LMR` 以及完整模型的差异。
3. 开展动态焦点率消融，比较固定 `r` 与动态 `r` 的性能和复杂度差异。
4. 开展多级 head 分配、注意力继承方式、全局分支接入位置等结构设计消融。
5. 通过局部放大图、误差图、多级注意力热力图和动态焦点率空间分布图解释模型行为。

## 6. 已有基础、预实验结果与可行性分析

### 6.1 已有研究基础

目前，围绕 `CAPA-SR` 的创新点定义、模块实现方案、实验设计方案和初步结果记录已经形成较为完整的研究基础。现有材料已经明确了模型命名、模块职责、核心超参数、Block 接口设计和实验验证路径。同时，当前工程目录中已经提供训练、测试与环境配置流程，为后续实现和复现实验提供了直接支撑。

此外，现有工作已对基线方法 `CATANet` 的设计逻辑和局限性进行了梳理，指出其在静态聚类中心、自适应性不足、强制等长分组引入噪声以及硬分配导致边界不稳定等方面仍有改进空间。这些分析为本课题提出 `CAPA-SR` 提供了清晰的问题来源与改进方向。

### 6.2 预实验结果

已有阶段性实验结果表明，`CAPA-SR` 在多个倍率上均表现出较强潜力。

1. 在 `x2` 设置下，模型以 `509K` 参数在 `Urban100` 和 `Manga109` 上分别达到 `33.71 / 0.9417` 与 `39.58 / 0.9793`，在复杂纹理场景中优于多数轻量级对比方法。
2. 在 `x3` 设置下，模型以 `582K` 参数在 `Urban100` 和 `Manga109` 上分别达到 `29.46 / 0.8765` 与 `34.63 / 0.9511`，整体表现接近或优于现有轻量级先进方法。
3. 在 `x4` 设置下，模型以 `567K` 参数、`46.1G FLOPs` 和 `85.1 ms` 延迟，在 `Urban100` 和 `Manga109` 上分别取得 `27.24 / 0.8179` 与 `31.55 / 0.9207`。

尤其值得注意的是，在 `x4` 设置下，`CAPA-SR` 相较于基线 `CATANet` 仅增加约 `32K` 参数，但 `Urban100` 提升 `0.37 dB`、`Manga109` 提升 `0.24 dB`，同时 `FLOPs` 从 `46.8G` 降至 `46.1G`，推理延迟从 `86.0 ms` 降至 `85.1 ms`。这说明所提方法并非简单以增加模型规模换取精度，而是在内容组织与连接筛选方式上取得了更优的性能-效率平衡。

消融实验同样支持课题设计的合理性。在统一 `x4 / DF2K / 300K` 设置下，从 `Model A` 到完整模型 `Model E`，模型在 `Urban100` 上由 `26.87 dB` 提升至 `27.24 dB`，在 `Manga109` 上由 `31.31 dB` 提升至 `31.55 dB`；动态焦点率在平均保留连接比例约 `43%` 的情况下优于固定稀疏率方案；`[2, 1, 1]` 的多级 head 分配优于其他分配方式；分层注意力继承优于无继承和共享单状态继承。这些结果表明，本课题提出的三个模块具有明确分工，并能形成有效协同。

### 6.3 可行性分析

从方法可行性看，本课题并非从零构建全新系统，而是在已有轻量级超分辨率框架和工程实现基础上进行模块化扩展，技术路线较为清晰。`DPR`、`PFSA` 和 `LMR` 在设计上彼此耦合但接口明确，便于分阶段实现与验证。

从实验可行性看，现有训练流程、测试脚本和模型配置已经具备，可直接支持主实验与消融实验运行。已有预实验结果也证明了该思路具有可落地性，说明本课题不仅具备理论动机，而且具备较强的工程实现基础。

从研究可验证性看，本课题已规划完整的主实验、消融实验、复杂度分析与可视化分析，能够从定量、定性和解释性多个维度验证研究结论。因此，整体上本课题具有较高可行性。

## 7. 预期创新点

本课题的创新点主要体现在以下三个方面：

1. 提出动态语义原型路由机制，以图像自适应方式组织长程内容相似 Token，突破固定窗口和静态中心带来的交互限制。
2. 提出渐进聚焦稀疏注意力机制，将当前层内容相似性与前序注意力先验联合建模，并引入内容感知动态焦点率，实现更有效的关键连接筛选。
3. 提出低到高多级协同重建机制，将结构恢复、边界过渡和纹理补偿纳入统一递进链路，使轻量级超分辨率模型更符合实际重建规律。

进一步看，这三点创新并非彼此孤立，而是共同构成一种围绕“内容组织、连接筛选、层次重建”三位一体的轻量级超分辨率设计范式，这也是本课题相较于现有方法的重要区别所在。
