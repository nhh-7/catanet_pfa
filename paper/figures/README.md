# Figure Assets

This directory contains self-contained LaTeX snippets used by the paper.

- `fig_intro_pareto.tex`: performance-efficiency snapshot for the introduction.
- `fig_method_overview.tex`: architecture overview for `CAPA-SR`.
- `fig_ablation_gains.tex`: gain summary for the core ablation.

These figures are referenced directly from the LaTeX sections via `\input{...}`.
