```Shell
pip install -r requirements.txt

python setup.py develop

conda install -c nvidia cuda-compiler

pip install deepspeed
```

进入 CATANet-PFA 目录后，使用以下命令启动单卡训练。这里使用 torchrun 并指定单节点单进程：

```shell
# 训练 x2 比例 (从头训练)
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x2_scratch.yml --launcher pytorch

# 训练 x3 比例 (微调)
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x3_finetune.yml --launcher pytorch

# 训练 x4 比例 (微调)
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x4_finetune.yml --launcher pytorch
```

1. 单卡测试命令

训练完成后，模型权重将保存在 experiments/ 目录下的对应工程文件夹中。你需要将训练好的最佳权重（或者官方预训练权重）放置到 pretrained\_models/ 目录中，然后执行测试：

```Shell
# 测试 x2 比例
python basicsr/test.py -opt options/test/test_CATANet_x2.yml

# 测试 x3 比例
python basicsr/test.py -opt options/test/test_CATANet_x3.yml

# 测试 x4 比例
python basicsr/test.py -opt options/test/test_CATANet_x4.yml
```

测试结果的图像和评估指标（PSNR, SSIM）将会保存在 results/CATANet 目录中。
