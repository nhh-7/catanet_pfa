# CATANet-PFA 全流程

## 1. 创建环境

建议使用独立 Conda 环境：

```bash
conda create -n catanet-pfa python=3.9 -y
conda activate catanet-pfa
```

## 2. 安装依赖执行以下命令：

```bash
pip install -r requirements.txt
pip install deepspeed
python setup.py develop
```
说明：

1. `deepspeed` 必须安装，因为当前工程的 [options.py](/Users/hainan/WorkSpace/SR/CATANet-PFA/basicsr/utils/options.py) 在导入时会直接 `import deepspeed`，不安装会在训练或测试入口直接报错。
2. `python setup.py develop` 必须执行，否则可能出现 `No module named 'basicsr'`。

## 3. 环境检查

安装完成后，执行：

```bash
python -c "from basicsr.archs.catanet_arch import CATANet; print(CATANet(upscale=2))"
```
如果可以正常打印网络结构，说明依赖安装完成，环境没有明显问题。

## 4. 单卡训练

下面命令默认都在 `CATANet-PFA/` 目录下执行。

### 4.1 x2 训练

```bash
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x2_scratch.yml --launcher pytorch
```
### 4.2 x3 训练

```bash
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x3_finetune.yml --launcher pytorch
```
### 4.3 x4 训练

```bash
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CATANet_x4_finetune.yml --launcher pytorch
```
