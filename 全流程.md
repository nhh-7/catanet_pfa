# CAPANet 全流程

## 1. 创建环境

建议使用独立 Conda 环境：

```bash
conda create -n capanet python=3.9 -y
conda activate capanet
```

进入工程目录：

```bash
cd /Users/hainan/WorkSpace/SR/CAPANet
```

## 2. 安装依赖

执行以下命令：

```bash
pip install -r requirements.txt
pip install deepspeed
python3 setup.py develop
```

说明：

1. `deepspeed` 必须安装，因为当前工程的 [options.py](/Users/hainan/WorkSpace/SR/CAPANet/basicsr/utils/options.py) 在导入时会直接 `import deepspeed`，不安装会在训练或测试入口直接报错。
2. `python3 setup.py develop` 必须执行，否则可能出现 `No module named 'basicsr'`。

## 3. DeepSpeed CUDA 环境修复（关键）

如果出现 `MissingCUDAException: CUDA_HOME does not exist`，执行：

```bash
conda install -c nvidia cuda-nvcc -y
conda install -c nvidia cuda-cudart-dev -y

export CUDA_HOME=$CONDA_PREFIX
export PATH=$CUDA_HOME/bin:$PATH
export LD_LIBRARY_PATH=$CUDA_HOME/lib64:$LD_LIBRARY_PATH
```

## 4. 环境检查

安装完成后，执行：

```bash
python3 -c "import deepspeed; print('deepspeed ok')"
nvcc --version
python3 -c "from basicsr.archs.capanet_arch import CAPANet; print(CAPANet(upscale=2))"
```

如果可以正常打印网络结构，说明依赖安装完成，环境没有明显问题。

## 5. 单卡训练

下面命令默认都在 `CAPANet/` 目录下执行。

### 5.1 x2 训练

```bash
CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CAPANet_x2_scratch.yml --launcher pytorch
```

### 5.2 x3 训练

```bash

```

### 5.3 x4 训练

CUDA_VISIBLE_DEVICES=0 torchrun --standalone --nnodes=1 --nproc_per_node=1 basicsr/train.py -opt options/train/train_CAPANet_x4_finetune.yml --launcher pytorch
