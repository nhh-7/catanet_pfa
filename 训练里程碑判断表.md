# ML-PFA-CATANet 训练里程碑判断表

> 说明：本文件沿用历史文件名 `PFA-CATANet训练里程碑判断表.md`，正文已按新的 `ML-PFA-CATANet` 方案更新。

本文档用于在训练过程中快速判断 `ML-PFA-CATANet` 是否处于正常收敛轨道。这里默认使用的是新方案主配置：

- `ML-PFA-IASA`
- `head_split = [2, 1, 1]`
- `Dynamic Focus Ratio`
- `beta = 0.5`
- `High + Mid` 保留 `IRCA`

## 前提条件

以下区间建立在这些条件基本不变的前提下：

- 训练集为 `DF2K`
- LR patch size 为 `64 x 64`
- 总训练步数为 `500K`
- 学习率策略为 `cosine decay`
- 模型主体不再发生结构性变化

## 当前参考点

以下是按新方案首轮稳定训练得到的早期参考值：

- `2.5k iter / 625 step`：Set5 `35.52`，Set14 `31.70`
- `5k iter / 1250 step`：Set5 `36.72`，Set14 `32.45`
- `10k iter / 2500 step`：Set5 `37.29`，Set14 `32.88`

### 3. 100k iter / 25000 step

### 4. 200k iter / 50000 step

### 5. 300k iter / 75000 step

- 偏慢：Set5 `< 38.28`，Set14 `< 33.85`
- 正常：Set5 `38.36 ~ 38.55`，Set14 `33.92 ~ 34.12`
- 很好：Set5 `>= 38.58`，Set14 `>= 34.15`

### 6. 最终收敛附近

- 保守可接受：Set5 `38.30+`，Set14 `33.95+`
- 接近目标结果：Set5 `38.36 ~ 38.45`，Set14 `34.10 ~ 34.20`
- 超出预期：Set5 `> 38.50` 且 Set14 `> 34.22`

## 使用建议

后续判断时不要只看 Set5，还建议同时观察三件事：

1. Set14 是否跟着一起抬升。
2. Urban100 是否在 `50k` 以后继续增长。
3. 动态焦点率是否出现过早塌缩到下限的情况。

## 异常信号

- 如果 Set5 正常、Set14 偏慢，优先检查中频分支和 `gamma_1` 是否过弱。
- 如果 25k 以后提升很快变缓，优先检查动态 `r` 是否过于保守。
- 如果 Set5 上升但 Urban100 长期不涨，通常说明低频分支没有真正学到跨区域结构先验。
